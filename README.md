# Sistema de Funcionários em Java

Este projeto é uma aplicação Java Console, desenvolvida com foco em Programação Orientada a Objetos (POO), abordando conceitos como herança, polimorfismo, classes abstratas e encapsulamento.
O sistema trabalha com diferentes tipos de funcionários e aplica regras específicas de cálculo e aumento salarial.
O projeto foi estruturado utilizando o NetBeans, sendo indicado para fins educacionais e acadêmicos.


## ⚙️ Funcionalidades

* Criação de diferentes tipos de funcionários

* Cálculo de salários conforme o tipo

* Aplicação de aumento salarial

* Demonstração prática de herança e polimorfismo

* Execução simples via console


## Tecnologias Utilizadas

* Java
